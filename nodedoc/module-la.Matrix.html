Class: Matrix

la. Matrix

Represents a dense matrix (2d array), wraps a C++ object implemented in glib/base/ds.h.

Constructor

new Matrix(argopt)

Matrix
Parameters:
Name Type Attributes Description
arg module:la~matrixArg | Array.<Array.<number>> | module:la.Matrix <optional>
 Constructor arguments. There are three ways of constructing:
1. Parameter object module:la~matrixArg.
2. Nested array of matrix elements (row major). Example: [[1,2],[3,4]] has two rows, the first row is [1,2].
3. A matrix (copy constructor).
Source:
  • ladoc.js, line 28
Example
// create new matrix with matrixArg
var mat = new la.Matrix({"rows": 3, "cols": 2, "random": true}); // creates a 3 x 2 matrix with random values
// create a new matrix with nested arrays
var mat2 = new la.Matrix([[1, 7, 4], [-10, 0, 3]]); // creates a 2 x 3 matrix
