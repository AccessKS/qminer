analytics.MDS

Source: analyticsdoc.js:1531

Multidimensional scaling

Method

getParams()

new MDS([params])

Example 

// import analytics module
var analytics = require('qminer').analytics;
// construct a MDS instance
var mds = new analytics.MDS({ maxStep: 300, distType: 'Cos' });

Parameters

Name Type Optional Description

params

(Object or module:fs.FIn)

Yes

The parameters for the construction.

Values in params have the following properties:

Name Type Optional Description

maxSecs

number

Yes

The maximum time period to compute MDS of a matrix.

Defaults to 500.

maxStep

number

Yes

The maximum number of iterations.

Defaults to 5000.

minDiff

number

Yes

The minimum difference criteria in MDS.

Defaults to 1e-4.

distType

string

Yes

The type of distance used. Available types: "Euclid", "Cos", "SqrtCos".

Defaults to "Euclid".

// import analytics module
