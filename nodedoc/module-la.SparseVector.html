Class: SparseVector

la. SparseVector

Represents a sparse vector.

Constructor

new SparseVector(argopt, dimopt)

Sparse Vector
Parameters:
Name Type Attributes Description
arg Array.<Array.<number>> | module:la.SparseVector <optional>
 Constructor arguments. There are two ways of constructing:
1. Nested array of vector elements. Example: [[0,2],[2,3]] has two nonzero values, first value is 2 at position 0, second value is 3 at position 2.
2. A sparse vector (copy constructor).
dim number <optional>
 Maximal length of sparse vector. It is only in combinantion with nested array of vector elements.
Source:
  • ladoc.js, line 210
Example
// create new sparse vector
var spVec = new la.SparseVector([[0, 1], [2, 3], [3, 6]]); // sparse vector [1, 0, 3, 6]
// create new sparse vector with dim
var spVec2 = new la.SparseVector([[0, 1], [2, 3], [3, 6]], 5); // largest index (zero based) is 4
