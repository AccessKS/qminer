sort(argopt) → {module:la.IntVector}

Sorts the vector (in place operation).
Parameters:
Name Type Attributes Description
arg module:la~intVectorCompareCb | boolean <optional>
 Default is boolean and true.
Source:
  • ladoc.js, line 1519
Returns:
Self
1. Vector sorted in ascending order, if arg is boolean and true.
2. Vector sorted in descending order, if arg is boolean and false.
3. Vector sorted by using the comparator callback, if arg is a module:la~intVectorCompareCb.
Type
module:la.IntVector
Example
// create a new vector
var vec = new la.IntVector([-2, 1, 3]);
// sort ascending
vec.sort(); // sorts to: [-2, 1, 3]
// sort using callback
vec.sort(function(arg1, arg2) { return Math.abs(arg1) - Math.abs(arg2); }); // sorts to: [1, -2, 3]
