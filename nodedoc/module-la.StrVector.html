sort(argopt) → {module:la.StrVector}

Sorts the vector (in place operation).
Parameters:
Name Type Attributes Description
arg module:la~strVectorCompareCb | boolean <optional>
 Sort callback or a boolean ascend flag. Default is boolean and true.
Source:
  • ladoc.js, line 1279
Returns:
Self.
1. Vector sorted in ascending order, if arg is boolean and true.
2. Vector sorted in descending order, if arg is boolean and false.
3. Vector sorted by using the comparator callback, if arg is a module:la~strVectorCompareCb.
Type
module:la.StrVector
Example
var la = require('qminer').la;
// create a new vector
var vec = new la.StrVector(['asd', 'z', 'kkkk']);
// sort ascending
vec.sort(); // sorts to: ['asd', 'kkkk', 'z']
// sort using callback
vec.sort(function(arg1, arg2) { return arg1.length - arg2.length; }); // sorts to: ['z', 'asd', 'kkkk']
