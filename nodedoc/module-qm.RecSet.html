Properties

empty

Checks if the record set is empty. If the record set is empty, then it returns true. Otherwise, it returns false.

length

Returns the number of records in record set.

store

Returns the store, where the records in the record set are stored.

weighted

Checks if the record set is weighted. If the record set is weighted, then it returns true. Otherwise, it returns false.

Keeps only the records with a specific value of some field.

Parameters

Name Type Optional Description

fieldName

string

 

The field by which the records will be filtered.

minVal

(string or number)

 


1. Is a string, if the field type is a string. The exact string to compare.
2. Is a number, if the field type is a number. The minimal value for comparison.
3. TODO Time field

maxVal

number

 

Only in combination with minVal for non-string fields. The maximal value for comparison.
Returns

module:qm.RecSet Self. Containing only the records with the fieldName value between minVal and maxVal. If the fieldName type is string, it contains only the records with fieldName equal to minVal.
